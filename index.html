  • Intro

    This is super easy way to bring HTML5 audio player to your Foundation based project. Take a look to README and the source of this page. This tab is also sandbox for Foundation Player API. There is the player global variable. Fill free to experiment in your browser's JavaScript console.

  • Playback control
    Method Description Returns
    playPause() This method toggles play or pause state of player. player object
    play() Starts player instance. player object
    pause() Pauses player. player object
    seekToTime() Seeks to exact position. Call player.seekToTime('00:42') to move to 00:42. player object
    seekPercent() Seeks to percentage. This code player.seekPercent(0.5) moves to 50%. player object

    Is's possible to use data-seek-to-percentage and data-seek-to-time attributes with links. In order to enable feature just passed {useSeekData: true} option to player.

    NOTE: The player's instance which was initialized the latest will handle links in the document. Any player which was initialized before will not handle data links. This option is safer to use when there is only one player on the page.

    <a data-seek-to-percentage='0.5'>50%</a> will create link which moves playhead to half position after clicking on it.

    <a data-seek-to-time='00:42'>00:42</a> will create link which moves playhead to 00:42 position after clicking.

    Please note default click action for links will be prevented.

  • Look and feel
    Method Description Returns
    togglePlayerSize() This method toggles player size. Method returns current size of player which was set. 'small' | 'normal'
    setPlayerSize(size) This method sets player size to one of possible values: 'small' or 'normal' . Player has 'normal' size by default. player object
